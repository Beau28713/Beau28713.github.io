Overview

Utilizing GitHub's API at "https://api.github.com/users/{username}/{tag}", the program retrieves a user's followers and those they are following, storing this data in JSON files. Users have the option to view who is following them or whom they are following.

How it works

To check who you are following but are not followed back:

  1. --username [username] --tag notfollowingme       return who you are following but are not following you back
  2. --username [username] --tag followingme        return who is following you but you are not following back
  3. --username [username] --tag getdata        retrieve users followers and following via GITHUB API and save to json file for parsing
